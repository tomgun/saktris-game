# F-0005: GameLoopAndTurns

## Acceptance Criteria

### AC1: Turn alternation
- White moves first
- Turns alternate between White and Black

### AC2: Turn flow
- Each turn: player either places piece OR moves piece (not both)
- If piece must be placed, player must place before moving

### AC3: Check detection
- Game detects when a king is in check
- "Check!" status displayed
- Player in check must escape check on their turn

### AC4: Checkmate detection
- Game detects when king is in check with no legal escape
- Game ends with winner announcement
- "Checkmate!" status displayed

### AC5: Stalemate detection
- Game detects when player has no legal moves but is not in check
- Game ends in draw
- "Stalemate!" status displayed

### AC6: Game over handling
- Game stops accepting moves after checkmate/stalemate
- Winner (or draw) clearly displayed
- New game can be started

### AC7: Turn indicator
- Current player's turn clearly displayed
- Visual indicator (color) shows whose turn
