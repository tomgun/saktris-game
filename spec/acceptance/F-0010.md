# F-0010: ArrowDrawing

## Acceptance Criteria

### AC1: Arrow creation
- Right-click and drag creates arrow
- Arrow starts from any piece

### AC2: Movement validation
- Arrows only go to valid destinations for piece type
- Invalid arrows shown in different color

### AC3: Knight L-shape
- Knight arrows show L-shaped path
- Path follows actual knight movement

### AC4: Arrow persistence
- Arrows remain until left-click
- Multiple arrows can be drawn

### AC5: Web export
- Context menu disabled in browser
- Arrow drawing works in web build

## Test Notes
- Visual/manual testing
- Test in both desktop and web exports
