# F-0014: BishopPlacementRule

## Acceptance Criteria

### AC1: Opposite color enforcement
- Cannot place bishop on same color square as existing bishop
- Both light-squared bishops blocked
- Both dark-squared bishops blocked

### AC2: Visual feedback
- Invalid bishop placement shows red tint
- Valid placements show normal

### AC3: Per-player rule
- Rule applies per player (White and Black separate)
- Opponent's bishops don't affect your placements

## Test Notes
- Unit tests verify placement logic
- Manual testing for visual feedback
