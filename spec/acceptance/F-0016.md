# F-0016: TimerMode

## Acceptance Criteria

### AC1: Chess Clock
- Per-player countdown timer
- Time decrements only during active player's turn
- Timer pauses during opponent's turn

### AC2: Time Controls
- Bullet (1+0, 2+1)
- Blitz (3+0, 3+2, 5+0, 5+3)
- Rapid (10+0, 10+5, 15+10)
- Classical (30+0, 30+20)
- Custom time control option

### AC3: Time Display
- Both players' remaining time visible during game
- Low-time warning (visual indicator when time is critical)
- Time formatted as mm:ss or ss.s for low time

### AC4: Time Expiration
- Game ends when a player's time reaches zero
- Opponent wins on time (unless insufficient material)
- Clear "time out" message displayed

### AC5: Increment Support
- Time increment added after each move (if configured)
- Increment applied correctly to the moving player

## Test Notes
- Manual testing with various time controls
- Verify timer accuracy over extended games
- Test time-out detection
