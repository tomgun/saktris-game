# F-0018: MultiplePieceSets

## Acceptance Criteria

### AC1: Standard Flat Pieces
- CBurnett-style flat 2D pieces available
- All piece types rendered (King, Queen, Rook, Bishop, Knight, Pawn)
- Both white and black sides

### AC2: Spatial 3D-Style Pieces
- 3D-style piece sprites available as alternative
- Shader-based 3D appearance (not actual 3D models)
- All piece types rendered for both sides

### AC3: Piece Set Selection
- Setting to choose between piece sets
- Selection persists between sessions
- Pieces update immediately on selection change

### AC4: Theme Integration
- Piece sets work with both Classic and Retrofuturistic visual themes
- Neon glow shader applies to pieces in Retrofuturistic theme
- Pieces scale correctly at all board sizes

## Test Notes
- Switch between piece sets in Settings
- Verify all pieces render correctly in each set
- Test with both visual themes
