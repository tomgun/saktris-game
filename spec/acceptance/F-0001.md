# F-0001: GodotProjectSetup

## Acceptance Criteria

### AC1: Project structure exists
- project.godot file exists and is valid Godot 4.5+ format
- Standard folder structure: src/, assets/, tests/

### AC2: Main scene loads
- Main scene (src/main.tscn) loads without errors
- Game manager initializes properly

### AC3: Autoloads configured
- Settings autoload available globally
- AudioManager autoload available globally
- ThemeManager autoload available globally

### AC4: Test framework available
- GUT (Godot Unit Test) or equivalent test framework installed
- Test runner script exists (tests/run_tests.gd)

## Test Notes
- Run `godot --headless --quit` to verify project loads
- Check project.godot for autoload configuration
