# F-0002: ChessBoardDisplay

## Acceptance Criteria

### AC1: 8x8 board renders
- Board displays 8 columns (a-h) and 8 rows (1-8)
- Squares alternate between light and dark colors

### AC2: Coordinate labels visible
- Column letters (a-h) displayed at bottom
- Row numbers (1-8) displayed on left side

### AC3: Responsive layout
- Board resizes appropriately when window is resized
- Pieces scale with square size

### AC4: Piece display
- All six piece types render correctly (King, Queen, Rook, Bishop, Knight, Pawn)
- White and black pieces visually distinguishable

### AC5: Click-to-move works
- Clicking a piece selects it (visual highlight)
- Clicking a valid destination moves the piece
- Clicking invalid destination deselects piece

### AC6: Drag-and-drop works
- Pieces can be dragged to new positions
- Invalid drops snap piece back to original position
- Drag provides visual feedback

## Test Notes
- Visual testing required - launch game and interact with board
- Test on multiple screen sizes/resolutions
