# F-0020: Game Rules Screen

## Acceptance Criteria

### AC1: Main Menu Link
- "Rules" or "How to Play" button visible on main menu
- Button styled consistently with other menu buttons
- Clicking opens the rules screen

### AC2: Rules Content Display
- Shows Saktris rules (from docs/SAKTRIS_RULES.md or similar content)
- Key sections: Overview, Piece Arrival, Placement, Victory Conditions
- Text is readable (appropriate font size, contrast)
- Content scrollable if longer than screen

### AC3: Navigation
- Clear "Back" or "Close" button to return to main menu
- ESC key also closes the rules screen
- No game state affected by viewing rules

### AC4: Mobile Support
- Rules screen works on mobile (responsive layout)
- Touch scrolling works
- Back button easily tappable (44px+ touch target)

## Test Notes
- Verify all rule sections are present and accurate
- Test scroll behavior on both desktop and mobile
- Ensure returning to menu works correctly
