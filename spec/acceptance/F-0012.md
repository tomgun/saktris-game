# F-0012: Player Name System

## Summary
Players can set a name that persists locally, is editable from a settings screen, and is used in online multiplayer. If not set when entering online play, the player is prompted.

## Acceptance Criteria
- [ ] Player name stored in Settings autoload and persisted to user://settings.json
- [ ] Settings menu accessible from main menu with name editing (max 20 chars)
- [ ] Save button persists name; Back returns to main menu
- [ ] Play Online prompts for name if not set (NamePanel)
- [ ] Name persists across restarts
- [ ] If name already set, Play Online skips prompt and goes straight to lobby
