# Asset Attribution

This file tracks all external assets used in Saktris and their licenses.

## Requirements

All assets must have licenses compatible with proprietary software:
- MIT
- Apache 2.0
- BSD
- CC0 (Public Domain)
- CC-BY (with attribution)
- CC-BY-SA (with attribution, share alike)
- Purchased/licensed commercial assets

**NOT ALLOWED:**
- GPL
- AGPL
- CC-NC (Non-Commercial)
- Assets without clear licensing
